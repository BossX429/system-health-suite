🔧 TROUBLESHOOTING GUIDE:

"Admin privileges required" error
  → Right-click batch file and select "Run as Administrator"

"Task Scheduler not updating"
  → Run: WORKFLOW_LAUNCHER.bat → [T] to check status
  → Verify Task Scheduler service is running

"Email won't send"
  → Check SMTP server address is correct
  → For Gmail: Use App-specific password, not regular password
  → Run: WORKFLOW_LAUNCHER.bat to reconfigure

"Dashboard shows no data"
  → Run GENERATE_BASELINES.bat first
  → Then run FULL_TEST.bat
  → Then open dashboard again

"Need to start over"
  → Delete: HealthOptimization.config.json
  → Delete: email.config.json
  → Run: WORKFLOW_LAUNCHER.bat [A] again
